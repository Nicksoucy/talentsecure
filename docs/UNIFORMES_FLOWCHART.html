pour 24 heures ouvrables avant la date butoir

(valide 7 jours)
→ Agent
🔄 SURVEILLANCE AUTO (toutes les heures)
Pendant la détention, le système vérifie automatiquement :
• Stock bas ou à zéro
• Lots de lavage bloqués
• Liens de signature qui expirent
• Plusieurs remises actives au même agent
→ Admin (cloche)
📧 T-24h OUVRABLES
"Rappel — retour d'uniforme dans 24h ouvrables"
→ rh@xguard.ca (+ cloche)
📧 DATE BUTOIR DÉPASSÉE
« ⚠️ Uniforme NON retourné — prélèvement requis $X »
→ rh@xguard.ca + paie@xguard.ca
📱 SMS via GoHighLevel
Lien sécurisé pour signer le retour
→ Agent
🔔 NOTIF DANS L'APP (cloche)
« Lot de lavage créé — N pièces à laver »
→ Admin (cloche)
📧 EMAIL D'ALERTE
« Retour avec pièces endommagées »
→ Admin + rh@xguard.ca
🔄 LOT OUVERT
Un seul lot ouvert est actif à la fois.
Tous les retours « Bon » s'y ajoutent automatiquement
jusqu'à son envoi.
⚡ AUTO À L'ENVOI
Dès qu'un lot est envoyé au lavage :
→ Une nouvelle batch ouverte est créée automatiquement
pour accueillir les prochains retours.
📧 SI LOT BLOQUÉ PLUS DE 7 JOURS
Alerte automatique si le fournisseur ne renvoie pas le lot
→ Admin (email + cloche)
📧 INSPECTION À FAIRE
« Lot revenu du lavage — à inspecter »
→ Admin (email + cloche)
✅ 2 OPTIONS À LA RÉCEPTION
Option A (cas normal) :
1 clic « Tout marquer Bon » → toutes les pièces retournent au stock.

Option B (si problème) :
Inspection pièce-par-pièce :
🟢 Bon → retour stock disponible
🔴 Endommagé → poubelle (sortie définitive)
💰 DETTE AGENT
Coût pièce × qty = $X
À prélever sur dernière paie
📧 → rh@xguard.ca + paie@xguard.ca
📧 SI DETTE >30j
Cron : relance auto si pas réglé
→ rh@xguard.ca + paie@xguard.ca
🔄 RETOUR AU STOCK DISPONIBLE
Quand l'inspection est finalisée, les pièces marquées Bon retournent automatiquement dans le stock disponible et peuvent être remises à un autre agent.
🚪 FIN D'EMPLOI (cas spécial)
Quand RH clique « Clôturer fin d'emploi » sur la fiche agent :
→ Toutes les pièces non retournées sont marquées comme telles
→ La dette est calculée automatiquement (somme des coûts de remplacement)
→ La remise est fermée définitivement
📧 → rh@xguard.ca + paie@xguard.ca + cloche
